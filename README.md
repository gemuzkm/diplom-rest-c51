#Data analytics

Data assembly and analysis. Generation of recommendations based on the processed data.

## Technology stack:
- Java 11
- Spring 5
- Spring Boot
- Database (MySQL and H2)
- Spring Data JPA
- Bean Validation
- Hibernate Validator
- Logging SLF4j
- Junit
- Maven
- Spring Security (JWT)
- Docker Compose
- Mapper (MupStruct)
- Swagger
